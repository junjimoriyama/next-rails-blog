### 💬 シンプルなSNSサイト

<br>

## 概要

メールアドレスとパスワードによる認証機能がある会員制のシンプルなSNS。

機能

・ブログの投稿、編集、削除

・他ユーザーの「お気に入り」登録

・他ユーザー投稿の「いいね」付け外し

・プロフィールの画像や名前の登録

・パスワード忘れの再設定

<br>
