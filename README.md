### 💬 シンプルなSNSサイト

## 概要

メールアドレスとパスワードによる認証機能がある会員制のシンプルなSNS。(JWT使用)

機能

・ユーザー登録、削除

・ブログ投稿、編集、削除

・他ユーザーの「お気に入り」登録

・他ユーザー投稿への「いいね」付け外し

・プロフィールの画像や名前の登録、変更

・パスワード忘れの再設定

<br>
